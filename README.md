# My-CTF-box

- Fork code from [racterub](https://github.com/racterub) and [staracg](https://github.com/staracg)

## Usage

```
git clone https://github.com/finn79426/CTF-box.git
cd CTF-box ; chmod +x install.sh ; ./install.sh
```

## Feature

- Install CTF-tools
- mkdir at `~/Tools`
- Git & GitHub setup
- Multi architecture setup
- Update rcfile
